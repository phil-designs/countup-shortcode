# Count Up Shortcode

Tags: jQuery, javascript, php
Requires at least: 3.6.0
Tested up to: 6.9.1
License: GPL2

## Description

A simple count up on scroll plugin.

## Tested on 
* Firefox 
* Safari
* Chrome
* Opera
* MS Edge

## Website 
http://www.phildesigns.com/

## Installation 
1. Upload ‘countup-shortcode’ to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Wrap your numbers with [count_up][c/ount_up] shortcode tags.
